# House Price Prediction and Locality Analysis in Hyderabad

## Project Overview

This project analyzes residential property data from Hyderabad and predicts house prices using Machine Learning.

## Technologies Used

- Python
- Pandas
- NumPy
- SQLite (SQL)
- Excel
- Matplotlib
- Scikit-Learn

## Dataset Features

- Location
- Property Type
- Area (Sq.ft)
- Building Status
- Rate Per Sq.ft
- House Price

Hyderabad housing dataset from Kaggle.

## Project Workflow

1. Load Hyderabad housing dataset
2. Clean and preprocess data
3. Store data in SQLite database
4. Generate Excel reports
5. Analyze locality-wise house prices
6. Visualize top localities
7. Train Random Forest Regression model
8. Predict house prices

## Outputs

- House_Report.xlsx
- House_Report_Summary.xlsx
- locality_prices.png
- house_price_model.pkl

The trained model file (house_price_model.pkl) was generated locally after training and is not included in the repository due to file size limitations.
